# Контрольная работа №1 - Технологии разработки серверных приложений

## Описание проекта
Проект представляет собой FastAPI-приложение, демонстрирующее основные возможности фреймворка:
- Создание маршрутов и обработка запросов
- Работа с Pydantic моделями и валидация данных
- Отправка HTML-страниц
- Обработка POST-запросов с JSON
- Хранение данных в памяти

## Запуск проекта

### Установка зависимостей

pip install fastapi uvicorn pydantic
